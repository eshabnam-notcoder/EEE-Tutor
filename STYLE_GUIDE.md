# EEE Tutor Style Guide

## Mission

EEE Tutor exists to explain Electrical Engineering concepts clearly and simply.

It is **not** an exam preparation website.

It is **not** a replacement for standard textbooks.

It is a concept clarification and quick revision platform.

---

# Design Principles

- Calm
- Clean
- Minimal
- Reader First
- Mobile Friendly

The interface should never distract from the content.

# Topic Structure

Every topic should follow this order.

# Topic Name

## Definition

A simple explanation in your own words.

---

## Diagram / Animation

Use an original diagram whenever it improves understanding.

---

## Formula

Include formulas only if applicable.

---

## Variable Meaning

Explain every variable clearly.

Example

V = Voltage (Volt)

I = Current (Ampere)

R = Resistance (Ohm)

---

## Explanation

Explain the concept step by step.

Assume the reader has forgotten the topic.

---

## Worked Example (Optional)

Only include if it improves understanding.

---

## Related Topics

List 3–5 related concepts.

---

# Writing Rules

Use simple English.

Prefer short sentences.

Avoid unnecessary jargon.

Explain symbols before using them.

Never copy textbook paragraphs.

Write everything in your own words.

---

# Diagrams

Create original diagrams.

Keep diagrams clean.

White background.

Simple labels.

Consistent colours.

No unnecessary decoration.
