# Superposition Theorem

The Superposition Theorem states that in a linear circuit with multiple independent sources, the voltage or current in any element is the sum of the effects produced by each source acting individually.

<img src="assets/animations/superposition.svg" alt="Superposition Theorem Animation">

---

## Formula

`Total Response = Response₁ + Response₂ + ... + Responseₙ`

---

## Key Points

- Applicable only to linear circuits
- Analyze one independent source at a time
- Replace voltage sources with short circuits
- Replace current sources with open circuits

---

## Simple Understanding

Find the effect of each power source separately, then add all the results together.
