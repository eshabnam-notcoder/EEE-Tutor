# Norton’s Theorem

Norton's Theorem states that any linear two-terminal network can be replaced by an equivalent current source in parallel with an equivalent resistance.

<img src="assets/animations/norton.svg" alt="Norton's Theorem Animation">

---

## Formula

`Equivalent Circuit = In || Rn`

Where:

- `In` = Norton Current
- `Rn` = Norton Resistance

---

## Key Points

- Dual of Thevenin's Theorem
- Simplifies circuit analysis
- Used for load current calculations
- Applicable only to linear circuits

---

## Simple Understanding

Replace a complex circuit with one current source and one parallel resistor that behaves the same as the original circuit.
