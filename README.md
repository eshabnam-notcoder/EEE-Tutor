# EEE Tutor

**Simple Concepts. Clear Revision.**

EEE Tutor is a free educational website designed to help Electrical and Electronics Engineering students revise core concepts quickly and clearly before exams.

---

## Features

- Markdown-based notes
- Clean lavender reading interface
- Mobile-friendly design
- Fast topic navigation
- Lightweight and Netlify-ready

---

## Adding a New Topic

Example:

```
content/basic-electrical/

    series-circuit.md
```

Open **section.html** and add a new topic card.

---

## Image Location

Store images inside

```
assets/images/
```

Example

```
assets/images/basic-electrical/
```

## Writing Notes

Use Markdown.

Example

```md

Ohm's Law states that...

## Formula

V = I × R

## Key Points

- Point 1
- Point 2
```
